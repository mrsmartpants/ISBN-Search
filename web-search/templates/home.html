About The Website :

This Website is designed to help searching & buying of books by just reviewing their details and comparing their prices on major book-selling websites in india. This single result page contain entire book details with price from different websites including links to buy that book on particular site if book is available there. On Result page you can search on "Author", "Publisher", "Publication Year" fields. It also contains Google Preview Button integrated to view book content if book is available for book preview on Google Books.
Result Contains Price of Books on following websites :

  • Flipkart
  • Infibeam
  • Bookadda
  • Homeshop18
  • Rediffbook
  • Crossword

Search Using Keywords

To search using keywords just enter the keywords which define your book topic.
For example : to search books on Mahatma Gandhi just enter Gandhi then select the book which suites you. Each Link contains the details.

Search using ISBN

Search using ISBN is comparatevely fast and smooth than keywords. Either enter the ISBN-13 or ISBN-10 in the search using ISBN field
For example to search book of "The Pragmatic Programmer: From Journeyman to Master 1st Edition" just enter ISBN-13 as '9780201616224' or ISBN-10 as '020161622X' in the isbn field then submit. If Book exists in the database you will see the result.
